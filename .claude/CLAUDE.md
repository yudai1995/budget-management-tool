## Claude Code 設定

このリポジトリの全規約・プロセス定義は **`.github/`** に一元管理されている（SSOT）。
作業開始前に以下のファイルを**必ず読み込み**、定義されたルールに従って作業すること。

| 用途 | 参照ファイル |
|------|-------------|
| AI エージェント必須プロセス・テスト要件・プロダクト判断基準 | `.github/ai-agent-workflow.md` |
| スプリントエージェントプロトコル（コマンド定義・ベロシティ） | `.github/sprint-protocol.md` |
| プロジェクト概要・ディレクトリ構造・命名規則・技術スタック | `.github/project-overview.md` |
| コーディング規約（TS/BE/FE/アイコン/禁止事項） | `.github/coding-conventions.md` |
| コミットメッセージ規約 | `.github/commit-message-instructions.md` |
| PR 生成ルール | `.github/pull-request-instructions.md` |
| マルチエージェント役割定義・Serena MCP ルール | `.github/multi-agent-roles.md` |
| レトロスペクティブテンプレート | `.github/retrospective-template.md` |
| ベロシティ実績ログ | `.github/velocity-log.json` |

矛盾が生じた場合は `.github/sprint-protocol.md` セクション 5「SSOT マップ」の優先順位に従うこと。

</important>
